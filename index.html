cnc-dxf-toolkit PublicÖffentlich

A DXF-prep library + CLI for CNC routing: cut start-point/lead-in control (keeps witness marks off the visible face), contour ordering, per-layer cut-depth, layer-name normalization, sheet-utilization reporting, and a configurable pre-flight validator — depth limits, stock-bounds, short segments, missing layers and machinability rules (e.g. 3- vs 5-axis). Layer names and machine rules are parameters, so it fits any shop.

Eine DXF-Aufbereitungs-Bibliothek + CLI für die CNC-Bearbeitung: Schnitt-Startpunkt-/Lead-in-Steuerung (hält Einstichspuren von der Sichtfläche fern), Kontur-Sortierung, Schnitttiefe pro Ebene, Ebenennamen-Normalisierung, Plattenausnutzungs-Auswertung und ein konfigurierbarer Pre-Flight-Validator — Tiefengrenzen, Plattengrenzen, zu kurze Segmente, fehlende Ebenen und Bearbeitbarkeitsregeln (z. B. 3- vs. 5-Achsen). Ebenennamen und Maschinenregeln sind Parameter und passen so in jeden Betrieb.

cnc-cycle-time PublicÖffentlich

A machine-learning cycle-time predictor that trains on your own data: point it at a folder of DXFs plus a telemetry/RTDB CSV of how long each job actually ran, and it learns to estimate the cut time of new parts. Generic geometry features (cut length, pierces, estimated rapid travel, entity/layer counts), flexible CSV ingestion (a duration column or start/end timestamps), and cross-validated model selection (Random Forest / gradient boosting) against a physics baseline. CLI: train / predict / evaluate.

Ein ML-Modell zur Zykluszeit-Vorhersage, das auf den eigenen Daten trainiert: man richtet es auf einen Ordner mit DXFs plus eine Telemetrie-/RTDB-CSV mit den tatsächlichen Laufzeiten, und es lernt, die Schnittzeit neuer Teile zu schätzen. Generische Geometrie-Merkmale (Schnittlänge, Einstiche, geschätzte Eilgang-Wege, Entitäts-/Ebenenzahlen), flexible CSV-Einlesung (Dauer-Spalte oder Start/Ende-Zeitstempel) und kreuzvalidierte Modellauswahl (Random Forest / Gradient Boosting) gegen eine Physik-Baseline. CLI: train / predict / evaluate.

Python · scikit-learn · ezdxf · pytest · CLI
github.com/PapaKoftes/cnc-cycle-time →

A full postprocessor for Biesse Rover routers: DXF → ISO 6983 / BPP / CIX with correct 5-axis kinematics and non-overridable safety limits. Built to replace per-machine vendor CAM licenses (€6k / 3-axis, €10k / 5-axis).

Ein vollständiger Postprozessor für Biesse-Rover-Maschinen: DXF → ISO 6983 / BPP / CIX mit korrekter 5-Achs-Kinematik und nicht überschreibbaren Sicherheitsgrenzen. Entwickelt, um die Hersteller-CAM-Lizenzen pro Maschine abzulösen (€6k / 3-Achs, €10k / 5-Achs).
